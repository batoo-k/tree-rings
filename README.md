# Tree Rings

Tree Rings is a Next.js prototype for a visual count-up timer inspired by tree rings.

Build inside the existing App Router project in this folder.
Do not create a separate root `index.html` prototype.

Important project docs live in `docs/`.
Read them before implementation, especially `docs/TASK_PLAN.md`, `docs/TIMER_LOGIC.md`, and `docs/VISUAL_RULES.md`.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.

Use SVG for the ring visualization and a system monospace font stack for the MVP.
